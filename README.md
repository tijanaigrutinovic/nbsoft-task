# nbsoft-task

Testni zadatak koji se sastoji od tri glavne funkcionalnosti:

1. **CV** - Stranica sa template-om za CV.
2. **Forma** - Forma za unos korisničkih podataka sa validacijom.
3. **Slider** - Interaktivni slider sa slikama.

## Tehnologije
- HTML5
- CSS3 (SCSS)
- JavaScript (jQuery)
- Bootstrap 5
- Slick Carousel
